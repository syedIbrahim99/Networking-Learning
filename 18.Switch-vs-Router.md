## **1Ô∏è‚É£ What is a Switch vs Router?**

| Feature         | Switch                              | Router                                       |
| --------------- | ----------------------------------- | -------------------------------------------- |
| OSI Layer       | Data Link Layer (Layer 2)           | Network Layer (Layer 3)                      |
| Function        | Connects devices **within a LAN**   | Connects **different networks**              |
| Address Used    | MAC addresses                       | IP addresses                                 |
| Traffic Type    | Local (inside same network)         | Local + Internet (between networks)          |
| Decision Making | Sends data only to intended device  | Chooses the **best path** to destination     |
| Example Use     | Office computers, printers, servers | Home/office router to connect LAN ‚Üí Internet |
| Security        | Helps local traffic management      | Can provide firewall and NAT services        |
