## What is a Public IP?

## Public IP

- **Public IPs are used on the internet**
- **They are globally unique**
- **They are allocated by ISPs and cloud providers**
- **ISPs and cloud providers receive public IPs from global registries like IANA and regional registries (APNIC, ARIN, RIPE, etc.)**

---

### Interview-ready one-liner (you can memorize this):

> **A public IP address is a globally unique, internet-routable address allocated by ISPs or cloud providers, which receive IP ranges from global and regional internet registries such as IANA and APNIC.**
